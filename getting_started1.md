
# OCaml简介第1部分

## 基本类型

### 整型（int）
#### 字首
* 二进制 0b
* 八进制 Oo
* 十六进制 0x
```ocaml
# 351;;
- : int = 351
# 12345;;
- : int = 12345
# 0o523;;
- : int = 339
# 0xffff;;
- : int = 65535
```

### 浮点数（float）
```ocaml
# 3.141;;
- : float = 3.1415
# 1.04e10;;
- : float = 10400000000.
# 25e-15;;
- : float = 2.5e-14
```
